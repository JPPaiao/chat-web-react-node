# chat-web-react-node
Copia do discord com chat em real time, client e server

## Techs
- Server
  - Node
  - Express
  - Typescript 
  - Socket.io (Para fazer a comunicação em tempo real)
  
- Client
  - React
  - Typescript
  - Socket.io.client (Para usar o socket no lado do client)
  - Tailwindcss (Para utilizar o css em classes)
